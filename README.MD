<h2>BuBuSSBot - Discord bot for private servers</h2>

<p>It's bot created for personal use on my discord server.</p>
<h1>How to use</h1>
<li>Create discord Bot on discord developer portal.
<li>Paste your bot Token into config.json file.
<li>Choose your prefix for commands, which will be prefix used for commands, and paste it into config file.
<li>Paste your guild ID(server ID) and owner ID, which you can acquire in developer mode in discord desktop app.
<li>Install required packages.
<li>Run your bot by executing run.py file.</li>
<li>To add new cogs, you need to create new .py file in folder source/cogs, it will be loaded with start of the bot.
<h1>Requirements</h1>
<li>Python 3.8+
<li>Discord account</li>
<p>Packages:</p>
<li>discord.py 
<li>apscheduler</li>
